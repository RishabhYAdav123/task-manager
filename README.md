# 🚀 Team Task Manager

A full-stack web application to manage projects and tasks with role-based access control.

---

## ✨ Features
- 🔐 Authentication (Admin & Member)
- 👥 Role-Based Access Control
- 📁 Project Management
- ✅ Task Assignment & Tracking
- 📊 Dashboard with statistics

---

## 🛠 Tech Stack
- Backend: Flask
- Database: SQLite
- Frontend: HTML, CSS
